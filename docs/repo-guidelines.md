# Repository-Guidelines

## Zielbild

`webapp-zentrale` ist die zentrale Uebersicht fuer Projekte, Webapps, Dokumentationen und Systemstatus.

## V1-Prinzip

- statisch und leichtgewichtig
- HTML, CSS, Vanilla JavaScript
- JSON als einfache Datenbasis
- keine Datenbank
- spaeter erweiterbar fuer PHP, Docker und Hosting

## Repo-Hygiene

- keine privaten Dateien
- keine lokalen Installer
- keine persoenlichen Dokumente
- keine unkontrollierten Build-Artefakte

## Community-Faehigkeit

Spaetere Mitwirkung soll moeglich sein durch:

- klare Struktur
- einfache Startlogik
- nachvollziehbare Dokumentation
- kleine, modulare Erweiterungen
