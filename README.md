# webapp-zentrale

Zentrale Startstruktur fuer neue Webapps, Projekte, Tools, Dokumentationen und Systemuebersichten.

## Ziel

Die `webapp-zentrale` soll als leichtgewichtige Hauptoberflaeche dienen fuer:

- Dashboard und Projektuebersicht
- Webapps und Entwicklungsbereiche
- Docker- und Infrastrukturbezug
- Dokumentationen und Referenzen
- Aufgaben, Roadmaps und Systemstatus

## Technische Basis V1

- HTML
- CSS
- Vanilla JavaScript
- JSON-Datenhaltung
- modulare Seitenstruktur

Keine Datenbank in V1.

## Projektstruktur

```text
webapp-zentrale/
  assets/
    css/
    js/
    img/
  data/
  pages/
  modules/
  docs/
  archive/
  backup/
  index.html
  README.md
  CONTRIBUTING.md
  LICENSE
```
